1. O QUE E O PROJETO
================================================================================

Nome:        UseCanuto
Tipo:        E-commerce de moda feminina
Inspiracao:  Maria Filo (mariafilo.com.br) — layout elegante, minimalista
Objetivo:    Loja virtual para promocao e uso comercial real

O UseCanuto e uma loja online completa com:
  - Pagina inicial animada e profissional
  - Catalogo de produtos por categoria (Vestidos, Blusas, Calcas, Saias, etc.)
  - Carrinho de compras com drawer lateral
  - Checkout com Cartao e PIX (modo demo ou Stripe real)
  - Painel administrativo completo
  - Conteudo da home 100% editavel pelo admin
  - Upload de fotos de produtos na nuvem (Supabase Storage)
  - Paginas de ajuda, contato, WhatsApp flutuante
  - Deploy automatico no Netlify a cada git push

================================================================================
2. TECNOLOGIAS UTILIZADAS
================================================================================

Frontend:
  - Next.js 16 (App Router)
  - React 19
  - TypeScript
  - Tailwind CSS 4
  - Framer Motion (animacoes)
  - Lucide React (icones)

Backend / API:
  - Next.js API Routes (serverless no Netlify)
  - Supabase JavaScript Client (@supabase/supabase-js)
  - Supabase Storage (fotos de produtos + conteudo do site)
  - JWT + bcryptjs (autenticacao admin)
  - Stripe (pagamentos — opcional)

Banco de dados:
  - Supabase (PostgreSQL na nuvem)
  - Tabelas: Admin, Category, Product, Order
  - Storage bucket: product-images (fotos + config/site-content.json)

Hospedagem:
  - Netlify (producao) — https://usecanuto.netlify.app
  - GitHub (codigo) — ScoobNoobDoo/usecanuto

Fontes:
  - Cormorant Garamond (titulos)
  - Jost (corpo do texto)

Legado (apenas desenvolvimento local, NAO usado em producao):
  - Prisma + SQLite/PostgreSQL local (seed e schema de referencia)

6. FUNCIONALIDADES DO ADMIN (PAINEL COMPLETO)
================================================================================

DASHBOARD (/admin):
  - Listar todos os produtos
  - Ver categoria, tamanhos, cores de cada produto
  - Editar preco rapido inline
  - Ativar/desativar produto
  - Excluir produto
  - Botoes: Editar Site | Novo Anuncio | Sair

NOVO ANUNCIO — Wizard 8 etapas (/admin/novo-produto):
  1. Iniciar cadastro
  2. Titulo da peca
  3. Categoria (Vestidos, Blusas, Calcas, Saias, Acessorios) — OBRIGATORIO
  4. Tamanhos (PP, P, M, G, GG + personalizados)
  5. Cores (Preto, Branco, Bege, Navy + personalizadas)
  6. Imagens (upload galeria PC/celular → Supabase Storage)
  7. Descricao
  8. Preco, preco promocional, destaque na home

EDITAR PRODUTO (/admin/produto/[id]):
  - Categoria
  - Fotos (adicionar/remover via upload)
  - Preco e preco promocional
  - Descricao

EDITAR SITE (/admin/conteudo):
  - Faixa superior rolante (frete, PIX, cupom, etc.)
  - Nome da marca e animacao de entrada
  - Banner principal (imagem, titulos, descricao, botoes)
  - Barra de beneficios (4 icones: pagamento, frete, entrega, cupom)
  - Secao categorias (titulos + imagem de cada categoria)
  - Secao destaques (titulo e subtitulo)
  - Newsletter (titulo, descricao, mensagem de sucesso)
  - Salvar tudo com um clique
